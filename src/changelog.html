Changelog

November 2015

  • Added support for updating the URLs of tables in a web map.
  • Enabled option to "Full Copy" a feature service. This creates a clone of the original feature service in the destination account including data and styling.
  • Fixed some bugs with the sign in process.

July 2015

  • Added support for working with Scenes (Web Scenes, Scene Layers, etc).
  • Include optimized build of the ArcGIS Javascript API with the source. Makes for faster loading and much easier to take the app into disconnected environments.
  • Fixed a bug when copying hosted applications.

March 2015

  • Added support for signing into IWA and PKI secured portals.
  • Display existing content in destination folders.
  • Various bugfixes and design tweaks.

January 2015

  • New landing page.
  • Added OAuth for a better sign in experience and to support enterprise logins.
  • Added a search bar. You can now search for content in ArcGIS Online, your organization/Portal, or inside of your own content using keyword fields.

    Search is currently limited to 100 results sorted by number of views in descending order.

  • Added download link for files when inspecting content.
  • Edit JSON. Make tweaks to content you own or content owned by others if you are their administrator.

September 2014

  • Major UI overhaul.
  • Enabled copying of feature services (copies a reference to the item only).
